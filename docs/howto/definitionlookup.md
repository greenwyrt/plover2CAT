## StarDict dictionaries

StarDict dictionaries are a special dictionary format (see more at [Wikipedia](https://en.wikipedia.org/wiki/StarDict)). Many dictionaries are available online in StarDict format.

## Install within the plover2cat folder

1. Download a StarDict dictionary. 
2. Extract files as the StarDict files are usually in zip/tar files.
3. Open the Plover Config folder from Plover (**File > Open Config Folder**) and go to the `plover2cat` folder.
4. If a folder called `stardict` is not present, create one.
5. Place the extracted folder/files into the `stardict` folder.
