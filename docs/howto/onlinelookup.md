# Online Lookup

Lookup a word/phrase in online services.

```{seealso}
For searching for text within the transcript, see [](findreplace.md).
For searching for misspellings within the transcript, see [](spellcheck.md).
For searching for closely related steno outlines, see [](stenosearch.md).
For lookup up a dictionary definition for a word/phrase, see [](definitionlookup.md).
```
Plover2CAT provides menu items to search a selected word/phrase in the following online services:

- Merriam-Webster Online
- Oxford English Dictionary Online
- Wiktionary
- Wikipedia
- Google Search Engine
- DuckDuckGo Search Engine

To lookup:

1. Select a word/phrase in the transcript.
2. Go to **Tools** and then **Online Lookup**. 
3. Select the online service and your browser will open up.
